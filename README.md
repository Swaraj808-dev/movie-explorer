# Movie Explorer

## Description

This is a movie web app that uses the TMDB API to show movie data.
Users can search for movies and also explore movies based on their mood like happy, romantic, or action.

---

## Features

* Search movies
* Mood-based movie suggestions
* View movie posters and ratings
* Save favourite movies

---

## Tech Used

HTML, CSS, JavaScript (Fetch API)

---

## How to Run

Open index.html and add your TMDB API key in script.js.

---

## Note

The UI is designed as MoodFlix to give a mood-based movie browsing experience.
